# Open Stomp

An open-source, hackable guitar effects processor and MIDI foot controller.

Combines neural amp modeling, IR loading, a full LV2 effects chain, and
programmable MIDI control in a single pedalboard-friendly device — built
entirely from open-source software and off-the-shelf hardware.

## Project Status

**Phase: Research & Planning**

See [ARCHITECTURE.md](ARCHITECTURE.md) for the full design document.

## Goals

- Neural amp modeling (NAM / AIDA-X compatible models)
- Stereo IR / cabinet simulation loading
- Full effects chain (delay, reverb, chorus, compression, EQ, etc.)
- MIDI control (USB, DIN, Bluetooth)
- WiFi-based web UI for pedalboard design
- Physical footswitches, expression pedals, LED feedback
- Hackable and extensible — open source top to bottom
- Prototype target: under $500
